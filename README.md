# Rick & Morty Character Explorer

A small web app to search and explore characters from the Rick and Morty API.

I built this project to practice working with APIs, handling asynchronous data, and rendering content dynamically in the DOM without using frameworks.

## 🚀 Features
- Search characters by name
- Filter by status (alive, dead, unknown)
- Dynamic rendering of results

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript (ES6+)
- Fetch API

## ⚙️ How to run
Just open `index.html` in your browser.
